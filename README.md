# Liquid Sort Game Solver

Solver for [liquid sort puzzle](https://apps.apple.com/in/app/sortpuz-water-puzzles-games/id1560298214).

### How to run solver:

```
STRICT_MODE=true ts-node solve.ts
```

### How to view results:

```
STRICT_MODE=true ts-node show.ts
```
